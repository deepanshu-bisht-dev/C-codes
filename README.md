# 🔧 C-codes

A set of C programming exercises and examples covering essential topics like input/output, loops, arrays, functions, and problem-solving techniques.

---

## 📌 About This Repository

This repo is a working log of C fundamentals — small, focused programs written while building a strong base in the language, before moving on to more advanced data structures and problem-solving.

- 🎓 B.Tech CSE (AI/ML Specialization), Future University)
- 🛠️ Focus: Core C concepts — operators, control flow, functions, arrays, strings
- 📈 Goal: Build strong fundamentals through consistent, hands-on practice

---

## 🗂️ Programs

| File | Description | Concepts |
|------|-------------|----------|
| [bitwise1.c](./bitwise1.c) | Demonstrates bitwise AND, OR, XOR, left shift, and right shift operators. | Bitwise operators |
| [code2.c](./code2.c) | Calculates the square root of a number. | `math.h`, `sqrt()` |
| [code3.c](./code3.c) | A minimal example of defining and calling a function with no parameters. | Functions |
| [code4.c](./code4.c) | A function that accepts and prints a parameter. | Functions, parameters |
| [code5.c](./code5.c) | Prints a number pattern using nested `for` loops. | Nested loops |
| [deep.c](./deep.c) | Converts a length in kilometers to meters, feet, inches, and centimeters. | `scanf`/`printf`, unit conversion |
| [deepanshu.c](./deepanshu.c) | Multiplies two 2×2 matrices entered by the user. | 2D arrays, nested loops |
| [enum.c](./enum.c) | Demonstrates defining and using an `enum` for days of the week. | Enums |
| [nested.c](./nested.c) | Checks if a number is positive/negative and even/odd using nested conditionals. | Nested `if-else` |
| [nested2.c](./nested2.c) | Classifies a person as minor, adult, or senior citizen based on age. | Nested `if-else` |
| [new.c](./new.c) | Demonstrates core string functions: length, copy, compare, and concatenate. | `string.h`: `strlen`, `strcpy`, `strcmp`, `strcat` |
| [switch.c](./switch.c) | Prints the day of the week using a `switch` statement. | `switch-case` |

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/deepanshu-bisht-dev/C-codes.git
cd C-codes
```

Compile and run any file with GCC:

```bash
gcc deep.c -o deep
./deep
```

> On Windows, run `deep.exe` instead of `./deep`.
